# Проект "Асинхронное взаимодействие клиента и сервера через TCP соединение"

## Описание

Данный проект представляет собой пример асинхронного взаимодействия между клиентом и сервером через TCP соединение с использованием Python 3.10 и asyncio.

## Функциональность

- **Сервер**: Принимает соединения на TCP порту и обрабатывает сообщения от клиентов. С вероятностью 10% игнорирует полученные сообщения. В остальных случаях отправляет ответы на запросы клиентов.
- **Клиенты**: Подключаются к серверу и отправляют запросы с сообщениями в определенном формате.

## Формат сообщений

- **Клиент -> Сервер**: `[порядковый номер запроса] PING\n`
- **Сервер -> Клиент**: `[порядковый номер ответа] PONG ([порядковый номер запроса]) ([порядковый номер клиента])\n`
- **Сервер -> Клиент (keepalive)**: `[порядковый номер ответа] keepalive\n`

## Логирование

- **Сервер**: Записывает в лог файл дату и время получения запроса, текст запроса, время отправки ответа, текст ответа. Для проигнорированных сообщений вместо текста ответа указывается "(проигнорировано)".
- **Клиенты**: Записывают в свои логи дату и время отправки запроса, текст запроса, время получения ответа или таймаута, текст ответа или "(таймаут)".

## Запуск

```
python3 main.py
```
Программа работает в течение 5 минут и завершается сама.

## Исходный код и логи

Исходный код программы, а также логи клиентов и сервера доступны в соответствующих файлах в папках client и server.
